# Yandex Reviews Integration

Приложение на **Laravel + Vue** для интеграции с карточкой организации в **Яндекс Картах**.

---

## Функционал

- Авторизация пользователя (логин / пароль)
- Страница настроек с возможностью указать ссылку на организацию в Яндекс Картах
- Получение и вывод данных компании:
    - рейтинг
    - общее количество отзывов
- Загрузка отзывов из карточки организации
- Постраничная навигация
- Сортировка отзывов по новизне

---

## Стек

- **Laravel**
- **Vue**
- **MySQL / SQLite**
- **Sanctum** (авторизация)

---

## Установка

### 1. Клонировать репозиторий

```bash
git clone <repo_url>
cd project
```

### 2. Установить зависимости

```bash
composer install
npm install
```

### 3. Создать `.env`

```bash
cp .env.example .env
php artisan key:generate
```

### 4. Настроить базу данных и выполнить миграции

```bash
php artisan migrate
```

### 5. Запустить приложение

```bash
php artisan serve
npm run dev
```

---

## Использование

1. Авторизоваться в системе
2. Перейти на страницу настроек
3. Вставить ссылку на карточку организации в Яндекс Картах
4. После сохранения будут отображены:
    - рейтинг компании
    - количество отзывов
    - список отзывов с пагинацией

---

## Примечание

Проект предназначен для демонстрации интеграции Laravel-приложения с внешним сервисом и работы с отзывами организации.
